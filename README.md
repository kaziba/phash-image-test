phash-image-test
======

# Usage

    git clone https://github.com/kaziba/phash-image-test.git
    cd phash-image-test
    npm run setup
    npm run start

# Requirement

On Debian:

    sudo apt-get install cimg-dev libphash0-dev libmagickcore-dev
